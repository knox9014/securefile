⬇ 다운로드
왜 다운로드인가요? 풀 엔진은 당신의 컴퓨터에서 직접 실행됩니다 — 파일이 인터넷으로 전송되지 않아 완벽한 프라이버시를 지키면서도 최고 압축률을 냅니다. 가벼운 사용은 위 "바로 사용"의 브라우저 버전으로 충분합니다.

왜 SecureFile인가

상용 압축 도구의 핵심 성능에, 더 현대적인 보안을 더했습니다.

🛡️

완전한 프라이버시

WebCrypto로 브라우저 안에서만 처리. 파일이 인터넷으로 전송되지 않습니다.

🔑

강력한 암호화

AES-256-GCM + Argon2id 키 유도. 변조 탐지 내장으로 위·변조를 차단합니다.

🗜️

똑똑한 압축

스트리밍 gzip 압축. 암호화 ON/OFF 토글로 압축만도, 압축+암호화도 자유롭게.

📁

폴더·다중 파일

여러 파일이나 폴더 통째로. 자동으로 묶어 하나의 파일로 만듭니다.

⚡

설치 불필요

브라우저만 있으면 끝. 계정·로그인·다운로드 없이 즉시 사용.

🎲

보안키 생성

외우기 쉬운 약한 비밀번호 대신, 추측 불가능한 강력한 키를 즉석에서 생성.
