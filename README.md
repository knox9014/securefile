# 🔐 SecureFile

파일/폴더를 **압축 + 암호화**하여 안전하게 보관하는 도구입니다.

**만든 사람: [knox9014](https://github.com/knox9014)**

## 구성
- `index.html` — **브라우저 웹 앱 (설치 불필요, 가장 편함)** ⭐
- `securefile.py` — 핵심 엔진 + 명령줄(CLI)
- `securefile_gui.py` — 파이썬 그래픽 인터페이스(GUI)

## 웹 앱 (추천)
`index.html`을 더블클릭해서 브라우저로 열기만 하면 됩니다.
- 설치 불필요, 드래그앤드롭으로 파일 잠그고 풀기
- **파일이 브라우저 밖으로 절대 나가지 않음** (서버 없음, 내 컴퓨터에서만 처리)
- 키 유도: Argon2id(인터넷 연결 시) / PBKDF2(오프라인 자동 대체)
- 압축: gzip · 암호화: AES-256-GCM · 진행률 표시줄 포함
- **단일 파일 / 여러 파일 / 폴더 통째로** 지원 (여러 개는 .tar로 묶어 암호화)
- 복호화 시: 단일은 원본 그대로, 묶음은 `.tar`로 복원 (윈도우 11은 더블클릭으로 풀기)
- 결과 파일 확장자: `.sfw`

## 사용법

### GUI (제일 쉬움)
```bash
python securefile_gui.py
```
파일/폴더 선택 → 비밀번호 입력 → 암호화/복호화 버튼 클릭

### 명령줄
```bash
python securefile.py encrypt <파일 또는 폴더>   # 잠그기 → .sfz 생성
python securefile.py decrypt <파일.sfz>         # 풀기
```

## 작동 원리
```
입력(파일/폴더) → LZMA 압축 → 1MB 청크마다 AES-256-GCM 암호화 → .sfz
```

| 요소 | 방식 |
| --- | --- |
| 압축 | LZMA (무손실, 높은 압축률) |
| 암호화 | AES-256-GCM (표준 + 무결성 검증) |
| 키 생성 | PBKDF2-HMAC-SHA256, 200,000회 |
| 무결성 | 청크별 순번 인증 → 변조/잘라내기/순서변경 탐지 |

## ⚠️ 주의
`.sfz` 파일과 비밀번호를 **둘 다 잃으면 복구 불가능**합니다.
이것은 안전한 암호화의 본질적인 특성입니다.
